# FRONT_biblioteca_isaque

## Login e Cadastro (sem backend)

Este projeto contém páginas estáticas de **Login** e **Cadastro** para uma biblioteca escolar, usando **localStorage** para simular persistência.

### Arquivos
- `index.html` → tela de login
- `cadastro.html` → tela de cadastro
- `assets/styles.css` → estilos
- `assets/auth.js` → validação e persistência local

### Como testar
1. Abra `index.html` no navegador.
2. Clique em **Criar conta** e finalize o cadastro.
3. Volte em **Já tenho conta** e faça login.

> Observação: Os usuários ficam armazenados somente no navegador (localStorage).
